Turn Any YouTube Video into an ESL Lesson in 60 Seconds

2026-03-20
← Back to blog
🎬
Teachers paste a YouTube URL. 60 seconds later: a full interactive ESL lesson with shadowing, quizzes, and a cheat sheet.

❌ The Problem

ESL teachers spend 2-4 hours creating a single lesson from a YouTube video. They manually transcribe, pick vocabulary, write questions, format handouts. Most give up and just play the video in class with no structure.

💡 The Breakthrough

YouTube aggressively blocks scraping from cloud servers. We tried 4 methods:

🚫
❌ youtube-transcript library → blocked on Vercel ❌ InnerTube API → blocked on Vercel ❌ Direct caption scraping → IP-locked ❌ Client-side fetch → CORS blocked
✅
✅ Gemini fileData API → feed the YouTube URL directly to Gemini. Gemini WATCHES the video and extracts the transcript. No scraping. No CORS. Works on Shorts, Live, anything.

"Sometimes the best solution isn't fighting the wall — it's walking around it."

📊 The Pipeline

👩‍🏫
TEACHER FLOW (3 clicks) ━━━━━━━━━━━━━━━━━━━━ 1️⃣ Paste YouTube URL 2️⃣ Pick CEFR level (A1-C2) 3️⃣ Click Generate 🐜 Ant walks across screen with fun facts... ✅ Full lesson ready in ~60 seconds

🎯 What Gets Generated

Stage 1: Intro

  • Full embedded YouTube video
  • AI-generated overview + learning goals

Stage 2: Key Expressions

  • 2-column phrase cards with ▶️ play buttons
  • Each button plays the exact video clip where the phrase appears
  • Vietnamese translations + sentence context
  • Interactive matching game

Stage 3: Video Practice

  • Mini video player pinned on top (always visible)
  • 3-4 shadowing cards: listen to native speaker → repeat
  • 8-10 MCQ + fill-in-blank questions (one at a time)
  • 3-4 sentence ordering questions
  • 2-3 "phrases in context" questions
  • Total: 18-22 interactive items, all sequential

Stage 4: Cheat Sheet

  • All key phrases (English + Vietnamese)
  • Full vocabulary list with definitions
  • Communication tips from the video
  • Video player for rewatching

🔧 Technical Architecture

⚙️
YouTube URL ↓ Gemini 2.5 Flash (fileData API) ↓ transcript + timestamps Clean Transcript API ↓ sections, vocab, key phrases Generate Lesson API (Gemini) ↓ structured HTML with interactive components Supabase (storage) ↓ Lesson Viewer (Next.js) ↓ sequential quiz flow, TTS, video clips Student's phone 📱

📈 Impact

📊
⏱️ 2-4 hours → 60 seconds 📚 Works with ANY YouTube video 🎯 Adapts difficulty to CEFR level 💰 $0 cost (Gemini free tier) 🌐 Vietnamese + English bilingual

🎥 Long Video Support

For videos over 10 minutes, the system shows a section picker. Teachers select which parts to teach. Only those segments are used for lesson generation — keeping lessons focused and manageable.
