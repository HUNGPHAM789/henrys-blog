74 Commits in One Day: Building an EdTech App with an AI Pair Programmer

2026-03-20
← Back to blog
One human. One AI assistant. 74 commits. 4 lesson modes shipped. All in a single day.

The Problem

Building an EdTech platform solo is overwhelming. You need frontend, backend, AI integration, TTS, video processing, database design — and you need to ship fast before motivation dies. Traditional development: 1-2 features per week if you're lucky.

Before vs After OpenClaw

WITHOUT AI PAIR PROGRAMMER ━━━━━━━━━━━━━━━━━━━━ 1 week → 1-2 features Context switching between docs, Stack Overflow, testing Bugs take hours to trace No documentation gets written Manual deploys, manual testing
WITH OPENCLAW + JARVIS ━━━━━━━━━━━━━━━━━━━━ 1 day → 74 commits, 4 complete lesson modes Say what you want → AI writes, tests, deploys Bugs fixed in real-time during conversation Auto-documented in memory + Notion journal Auto-push to GitHub → auto-deploy to Vercel

How It Works

Step 1: Boss describes the feature

"Merge all question types into one practice stage — video on top, 1 question at a time"

Step 2: Jarvis plans + executes

  • Reads the codebase to understand current architecture
  • Edits the Gemini prompt (API route) to restructure the output
  • Updates the lesson viewer (frontend) with new CSS + JS
  • Commits + pushes to GitHub in one atomic change

Step 3: Vercel auto-deploys

Push to master → Vercel builds → live in ~2 minutes. Boss refreshes and tests.

Step 4: Iterate in real-time

"The matching game looks dull" → Jarvis redesigns with gradient bg, shake animation, pill score badge → pushed in 3 minutes.

Results

74 commits │ 4 lesson modes │ 22+ files restyled │ YouTube transcript pipeline │ All in ~14 hours
  • Standard mode: flip cards, matching games, quizzes, dialogues, speaking practice
  • Decision mode: branching roleplay with social score tracking
  • Video mode: YouTube URL → AI generates full interactive lesson with shadowing
  • IELTS mode: Part 1 & 2 generators (auto-committed)
  • Bubbly UI redesign across entire app
  • Animated ant loading screen

Key Takeaway

OpenClaw turns your AI assistant into a full development team. You describe what you want in plain language, and the agent handles code, testing, deployment, and documentation. The human stays in the driver's seat — making product decisions, not debugging semicolons.
